# 1. 小白机器人 UPS 电路

UPS 电路原理图如下，

<img src="https://raw.githubusercontent.com/Tz-slayer/image-bed/master/markdown/20250919100555-1758276355644.png"/>

充电管理芯片使用的是 ETA6002E8A，也就是开关型锂电池充电器，特性如下：
  
1. **充电管理功能**：能依据电池电压自动切换预充、快充、恒压充电模式。
2. **功率路径管理功能**：电池电量足够时，优先使用电池给系统供电；没有电池，系统也能靠外接适配器给系统供电。
3. **其他功能**：具有无电池检测功能，通过 STAT 引脚指示充电状态。

:::note
**快充电流计算**

快充电流的值由 ISET 和 GND 之间的电阻决定，计算公式为 $I_{BAT}=\frac{1000}{R_{ISET}}$

<img src="https://raw.githubusercontent.com/Tz-slayer/image-bed/master/markdown/20250919103411-1758278051824.png" style="margin:0px;width:30%;"/>

**充电状态指示**

STAT 引脚指示充电的状态

| 充电状态 | STAT 输出 |
| -------- | --------- |
| 正在充电 | 低电平 |
| 充电完成 | 高阻态|
| 超过安全充电时间 | 1Hz 频率闪烁|
| NTC 保护(温度过高) | 1Hz 频率闪烁|
| 没有电池 | 1Hz 频率闪烁|

**动态功率路径管理**
1. 当芯片接入外部电源适配器时，SYS 引脚优先从内部的 DC-DC 稳压器输出的电压为系统供电，而非直接使用电池。
2. 当外部电源断开时，芯片会自动切换到电池供电。
3. 即使不接电源，只要 IN 引脚接入外部电源，芯片就可以通过 SYS 引脚为系统供电。
:::

对应到电路图中，也就是当电池正在充电的时候，LED1 会亮灯，当充满电的时候，会熄灭。
